# 📚 JS Study

Repositório dedicado aos meus estudos em JavaScript, com foco em lógica de programação, estruturas de controle e fundamentos essenciais para desenvolvimento backend e frontend.

---

## 🚀 Objetivo

Este projeto tem como finalidade:

- Praticar lógica de programação
- Consolidar conceitos de JavaScript
- Evoluir na escrita de código limpo
- Simular cenários reais de desenvolvimento

---

## 🧠 Conteúdos abordados

- Variáveis e tipos de dados
- Operadores
- Estruturas condicionais (`if`, `else`)
- Operadores booleanos
- Entrada de dados via terminal
- Tratamento básico de input
- Boas práticas iniciais

---

## 🛠️ Tecnologias utilizadas

- JavaScript (Node.js)
- prompt-sync (entrada de dados via terminal)
